1. extern	外在變數		使區域變數變成全域變數 
2. Union    型別			將不同data types儲存在同一個記憶體空間的特殊自訂型別
3. inline 	行內函式		可以「建議」編譯器將之設定為「行內函式」（Inline function），如果建議被採納，則該函式會自動在呼叫點展現為程式碼
4. ==		比較			再指標時是比較記憶體位置，再使用一般的變數時是比較值
5. strcmp	C語言的字串比較	會比較字串長度與字串內容
6. int						有時候根據所在環境不同或是晶片不同，int之類的變數宣告大小會不一樣ex:arduino int 8bit  C int 16bit
